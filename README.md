# clearfix

Clear floated children of an element. Uses the [micro-clearfix](http://nicolasgallagher.com/micro-clearfix-hack/) by Nicolas Gallagher.

## Support

Everything and IE8+.

## Install

```
component install segmentio/clearfix
```

## Usage

```html
<ul class="clearfix">
  <li>Nav Item</li>
  <li>Nav Item</li>
  <li>Nav Item</li>
</ul>
```
